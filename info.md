# StokerCloud Custom Integration

Diese Integration liest Ihre StokerCloud Heizungsdaten und stellt sie in Home Assistant bereit.

## Installation

1. Fügen Sie dieses Repository in HACS als benutzerdefiniertes Repository hinzu
2. Installieren Sie "StokerCloud Custom"
3. Starten Sie Home Assistant neu
4. Fügen Sie die Integration hinzu: Einstellungen → Geräte & Dienste → Integration hinzufügen
5. Geben Sie Ihren StokerCloud Benutzernamen ein (z.B. `NBE-XXX`)

## Features

- ✅ Einfache Einrichtung - nur Benutzername erforderlich
- ✅ Alle Heizungsdaten als Sensoren
- ✅ Automatische Updates alle 60 Sekunden
- ✅ Deutsche Übersetzungen
